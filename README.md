# Chai CSS Engine 🚀

A lightweight utility-first CSS engine built using JavaScript.

## 🔥 Features

* Utility classes like `chai-p-10`, `chai-bg-red`
* Dynamic DOM parsing
* Inline style application
* Hover support (`chai-hover-*`)
* Supports layout utilities (flex, center)

## 🧠 How it Works

* Scans DOM after page load
* Detects classes starting with `chai-`
* Parses class patterns into key-value pairs
* Maps them to CSS styles
* Applies styles dynamically using JavaScript

## 🧪 Example

```html
<div class="chai-p-20 chai-bg-blue chai-hover-bg-red">
  Hover me 🔥
</div>
```

## ⚙️ Tech Stack

* JavaScript (DOM Manipulation)
